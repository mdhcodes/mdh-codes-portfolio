Portfolio

Spiral Twists

Photo by: Michelle Hartley

Website: www.spiraltwists.com

HTML5, CSS3, JavaScript, Flexbox

Spiral Twists is a digital gallery of my work as a potter. I chose the name for the many spirals found in nature especially those that form in the clay as it is wedged using a technique known as spiral wedging.

This is the first website I launched. Since then, I’ve refactored the code and added new functionality to include HTML5 and CSS3 elements along with JavaScript for interactivity and Flexbox for layout. I’ve enjoyed watching my work evolve and I look forward to future upgrades and new projects.

Random Quotes

Photo by: Patrick Tomasso

Link: Random Quotes

HTML5, CSS3, JavaScript, API

Random Quotes allows users to browse famous quotes and tweet their favorites.

JavaScript requests data from the Andrux Random Famous Quotes API which delivers a response that is parsed to extract the information needed to fulfill the requirements and user stories (what the user should be able to do with the application or website). JavaScript accesses this data and manipulates the DOM to dynamically populate the screen with the selected content.

Local Weather

Photo by: Patrick Fore

Link: Local Weather

HTML5, CSS3, JavaScript, API

Local Weather gathers and displays weather information based on your location. When the page loads, you'll see the current temperature, an image describing the weather, and a few other interesting facts. Click the button next to the temperature and the units will convert from ℃ to ℉ and back.

The freegeoip.net API determines your current location and the Open Weather Map API uses the information (latitude and longitude coordinates) to deliver the weather data for your area.

mdhCodes

Photo by: Jonathan Bean

Website: mdhCodes.com

HTML5, CSS3, JavaScript, jQuery, Flexbox

mdhCodes is an ongoing portfolio of my work as a developer. The highlight of building this site was programming the resume page.

It was developed using an HTML template to outline the structure of the page, JavaScript objects to store the data, and jQuery to access and append that data to the DOM. The Google Maps API provides a bird’s-eye view of my location along with great places to visit in the area.

About

Michelle Hartley

I’ve always been fascinated by computers and all that goes on behind the scenes to make software applications function effectively and efficiently.

After reading my first programming book and applying those principles to run scripts and create websites, I was hooked. I taught myself HTML, CSS, and web development best practices. I supplemented my reading with online coursework and continued to learn JavaScript and jQuery.

My background in education has honed my research and problem solving skills while my work as a potter nurtures my imagination and creativity. As a developer, I utilize these tools and many others that I’ve gathered along the way to explore the world inside the computer and transform ideas into reality through code.

I enjoy every moment of the development process. My love of computer programming propels me forward and my passion for learning drives me to dig deeper and reach higher every day.
