Portfolio

Spiral Twists

Spiral Twists

HTML5, CSS3, JavaScript, Flexbox

Spiral Twists is a digital gallery of my work as a potter.

Starting with a mobile first approach, all images were scaled down to improve performance and media queries were added to adjust the layout for different screen sizes. The links change appearance when a user hovers over them to indicate an interaction is possible. CSS3 transition and transform properties animate the gallery categories. JavaScript controls the display of the navigation bar by responding to the user when the menu button is clicked.

Random Quotes

Random Quotes

HTML5, CSS3, JavaScript, API

Random Quotes allows users to browse famous quotes and tweet their favorites.

JavaScript requests data from the Andrux Random Famous Quotes API which delivers a response that is parsed to extract the information needed to fulfill the project requirements (website functionality) and user stories (what the user should be able to do when interacting with an application or a website). JavaScript accesses this data and manipulates the DOM to dynamically populate the screen with the selected content.

Local Weather

Local Weather

HTML5, CSS3, JavaScript, API

Local Weather gathers and displays weather information based on your location. When the page loads, you'll see the current date, the time, your location, the temperature, an image describing the weather, and a few other interesting facts. Click the button next to the temperature and the units will convert from ℃ to ℉ and back.

The geolocation API determines your current location and the Weather Underground API uses that information (latitude and longitude coordinates) to deliver the weather data for your area.

Wikipedia Viewer

Wikipedia Viewer

HTML5, CSS3, JavaScript, API

Wikipedia Viewer is a great way to learn something new everyday. This application allows users to view random articles or search for specific information in the Wikipedia database. Users type the name of a subject, place, or person of interest in the search field, press enter or click the search button, and the app returns 15 related Wikipedia entries. The user can then read page snippets and click on a link to view a full page.

Wikipedia Viewer uses the MediaWiki API to access the Wikipedia database.
